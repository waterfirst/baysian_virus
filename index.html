1. 탐구 동기

코로나19 팬데믹은 전염병 예측의 중요성을 일깨워주었습니다. 기존 예측 모델은 초기 설정값에 크게 의존하지만, 베이지안 업데이트는 새로운 데이터가 들어올 때마다 예측을 개선합니다.

왜 베이지안 업데이트인가?

  • 실시간 예측 개선 가능
  • 불확실성 상황에서 효과적
  • 데이터가 쌓일수록 정확도 향상
  • 복잡한 수학 없이 직관적 이해 가능

탐구 목표

베이지안 업데이트를 활용하여 코로나 바이러스와 같은 전염병의 확산을 예측하는 시뮬레이션 모델을 개발하고, 다양한 상황에서의 예측 정확도를 분석합니다.

3. 문헌조사 - 전염병 예측 모델

SIR 모델 (감염자-회복자 모델)

전염병 확산을 설명하는 가장 기본적인 모델:

  • S: Susceptible (감염 가능 인구)
  • I: Infected (감염자)
  • R: Recovered (회복자/사망자)
기초감염재생산지수 (R0)

1명의 감염자가 평균적으로 몇 명을 감염시키는지를 나타내는 지수

R0 < 1

R0 > 1

4. 탐구 방법 - 가정 및 모델

단순화된 가정

  • 인구 1,000명의 가상 도시
  • 초기 감염자: 10명 (감염률 1%)
  • 검사 정확도: 90% (가정)
  • 매일 100명 검사 (가정)
  • 외부 요인 무시 (이동, 백신 등)

베이지안 업데이트 모델

매일 새로운 데이터로 감염률 예측 업데이트:

업데이트된 감염률 = (사전 감염률 + 신규 감염률) / 2

5. 탐구 방법 - 베이지안 업데이트

업데이트 프로세스

  1. 초기 감염률 설정 (사전 확률)
  2. 신규 감염자 데이터 수집 (가능도)
  3. 베이지안 업데이트 계산
  4. 업데이트된 감염률 저장 (사후 확률)
  5. 다음 날 예측을 위한 새로운 사전 확률로 사용

파이썬 구현 예시

def update_bayesian(prior, new_cases, tested):
    # 신규 감염률 계산 (가능도)
    likelihood = new_cases / tested

    # 베이지안 업데이트 (사후 확률)
    posterior = (prior + likelihood) / 2

    return posterior

11. 고찰 - 장점과 한계

장점

  • 새로운 데이터로 실시간 예측 개선 가능
  • 불확실성 관리에 효과적
  • 고등학교 수준에서 이해 가능한 수학적 모델
  • 초기 가정 오류를 빠르게 수정

한계

  • 검사 정확도, 검사 수 가정에 의존
  • 인구 이동, 백신 등 외부 변수 미반영
  • 단순화된 모델로 현실 복잡성 반영 부족
  • 장기 예측 정확도 제한적

본 시뮬레이션은 교육적 목적의 단순화된 모델입니다. 실제 전염병 예측에는 더 복잡한 모델과 다양한 변수들이 고려되어야 합니다.
